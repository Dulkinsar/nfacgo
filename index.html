Motivation & Problem Statement

  • Growing need: 15 million people suffer strokes annually worldwide, with many experiencing lasting mobility impairments
  • Current limitations: Traditional rehabilitation is resource-intensive, repetitive, and lacks personalization
  • Robotic solutions: Exoskeletons offer programmable movement assistance but often lack adaptability
  • Our focus: Integrating machine learning to create intelligent, adaptive rehabilitation systems that respond to individual patient needs in real-time

(Coser et al., 2024; Zhang et al., 2024)

Scenario and System Description

  • Wearable sensors: IMUs, EMG, and force sensors collect real-time movement data
  • ML processing: Neural networks analyze gait patterns and predict optimal assistance
  • Adaptive control: Reinforcement learning adjusts exoskeleton torque based on terrain and performance
  • Feedback loop: Continuous monitoring enables dynamic therapy adjustments

(Liu et al., 2024; Luo et al., 2023)

Research Objectives

  1. Develop an ML framework for real-time gait phase classification with >90% accuracy
  2. Implement adaptive control strategies using reinforcement learning
  3. Create predictive models for rehabilitation outcome forecasting
  4. Design assist-as-needed algorithms that reduce patient effort by 30%
  5. Evaluate system performance in clinical trials with 20 stroke patients

Methodology

  • Phase 1: Literature review & dataset collection (3 months)
  • Phase 2: Algorithm development (4 months)
    • Gait recognition using PSO-ELM
    • RL-based adaptive control
  • Phase 3: System integration & testing (3 months)
  • Phase 4: Clinical validation (2 months)

(Liu et al., 2025; Huang et al., 2025)
